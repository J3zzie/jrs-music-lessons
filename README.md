# JRS Music Lessons Template

A responsive Next.js + TypeScript website template inspired by the supplied black/orange JRS design.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Edit content

- Main page: `app/page.tsx`
- Text, lessons, reviews and benefits: `data/site.ts`
- Global styling: `app/globals.css`
- Contact form: `components/BookingForm.tsx`
- Footer details: `components/Footer.tsx`
- Original design reference: `public/reference/design-reference.png`
