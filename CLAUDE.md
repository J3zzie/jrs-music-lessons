# JRS Music Lessons — Claude Instructions

## Project
This is a Next.js App Router website written in TypeScript. Styling is in `app/globals.css` with no UI framework.

## Visual direction
- Match `public/reference/design-reference.png`.
- Keep the site dark, dramatic, rock-inspired, and high contrast.
- Primary accent: vivid orange.
- Use large condensed uppercase headings.
- Preserve generous spacing and responsive behavior.

## Code rules
- Keep components reusable and accessible.
- Do not remove working configuration.
- Avoid unnecessary packages.
- Keep editable content in `data/site.ts` where practical.
- Verify desktop and mobile layouts after changes.
- Run `npm run build` before finishing large changes.
